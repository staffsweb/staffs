Backgrounds are img elements not backgrounds, sized with object fit so they can take advantage of srcset if possible. Add an inline style [x/y] of object-position to set the focal point for the image, eg. style="object-position: 70% 20%" for an image with a focal point towards the top right.

Add a class of banner__bg--blurred to the banner__bg to always blur the background - backgrounds will by default be blurred only at small viewport widths.

Feature Tiles

London Colour Variants

These can be styled with a modifier to the feature-tiles__item, which will cascade to the title__highlight and pseudo-square (if the grid item has one)

Modifiers are in the $tile-accent-colors array in _main-ldn.scss, currently: accent-blue-ldn, accent-blue-deep-ldn, accent-magenta-ldn, accent-purple-deep-ldn, and accent-tef-gold.

NB: Use the modifier on this item, not the tile if you want the accent to cascade correctly to the pseudo-square, you don't need to add to the tile as well :)
