Banner

Backgrounds are img elements not backgrounds, sized with object fit so they can take advantage of srcset if possible. Add an inline style [x/y] of object-position to set the focal point for the image, eg. style="object-position: 70% 20%" for an image with a focal point towards the top right.

Add a class of banner__bg--blurred to the banner__bg to always blur the background - backgrounds will by default be blurred only at small viewport widths.

Cards

All types of cards work with the LDN skin, by default they use the primary colour, and accent modifiers relative to LDN can be applied.

These modifiers, cascade to the appropriate accented elements in different card types.

Like other LDN themed component variants, the modifiers are in a map in main-ldn.scss, consider this the source of truth, but at the time of typing, they're currently:

  • blue-ldn
  • blue-deep-ldn
  • magenta-ldn
  • purple-deep-ldn
For example: 
<div class="card  card--flex  card--magenta-ldn"></div>

Feature Tiles

London Colour Variants

These can be styled with a modifier to the feature-tiles__item, which will cascade to the title__highlight and pseudo-square (if the grid item has one)

Modifiers are in the $tile-accent-colors map in _main-ldn.scss, currently: accent-blue-ldn, accent-blue-deep-ldn, accent-magenta-ldn, accent-purple-deep-ldn, and accent-tef-gold.

NB: Use the modifier on this item, not the tile if you want the accent to cascade correctly to the pseudo-square, you don't need to add to the tile as well :)

As the map is overwritten, colours from the main site are not available unless explicitly added to the map in Sass.

Tiles

In the LDN theme, the default tile uses the $primary colour, as in the main theme.

These can be styled with a modifier, which will cascade to the title__highlight

Like other LDN themed component variants, the modifiers are in a map in main-ldn.scss, consider this the source of truth, but at the time of typing, they're currently:

  • accent-blue-ldn
  • accent-blue-deep-ldn
  • accent-magenta-ldn
  • accent-purple-deep-ldn
  • accent-tef-gold
For example: 
<div class="tile  tile--accent-blue-deep-ldn"></div>

As the map is overwritten, colours from the main site are not available unless explicitly added to the map in Sass.
